Pace counts Portuguese path sessions only (course-scoped day-over-day deltas — chess or other courses can't pollute it; freeform practice outside the path isn't counted). Lessons remaining are counted exactly from the course path — the declared sessions left in every unit through the end of Section 4 (unit 131, CEFR A2), minus the one optional session per unit that never gets done (per-unit counts come from the course path itself: sessions finished in each unit — day boundaries, other courses, and skipped units can't distort them). Everything is anchored to the latest snapshot — the page never counts a not-yet-snapshotted day as zero. Days that were skipped mid-window show as · and count as zero in the averages. Snapshots land nightly at 23:59.
